    Flexbox - float yoki position xossalari yordamisiz veb sahifada
moslashuvchan (flexible) layout yaratishda yordam beradi.
Boshqacha qilib aytganda, veb sahifadagi elementlarni joylashtirish
uchun xizmat qiladi. Flexbox 2 asosiy elementlardan tashkil topgan:
flex container va flex items

    Flex container - display xossasi uchun flex qiymatiga ega bo'lgan
elementga aytiladi

    Flex items - veb sahifamizdagi flex container ota-onaga ega bo'lgan
elementlar

    Flexbox model asoslari quyidagilardan tashkil topgan: flex
container, flex item, asosiy o'q (main axis), kesuvchi o'q (cross axis)

    Flexbox uchun ishlatilinadigan CSS xossalarimiz 2 guruhga bo'linadi:
flex container va flex items xossalari
